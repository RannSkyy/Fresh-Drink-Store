# FreshDrink Store ☕

FreshDrink Store is a modern beverage e-commerce web application built using vanilla HTML, CSS, and JavaScript. This project demonstrates dynamic product rendering, a functional shopping cart system, local storage persistence, and a responsive user interface.

---

## 🚀 Features

- Dynamic product rendering from JavaScript data
- Add multiple items to cart
- Automatic total price calculation
- Remove items from cart
- LocalStorage support (cart persists after refresh)
- Simulated payment processing
- Responsive layout for desktop and mobile
- Clean and structured project architecture

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox & Responsive Design)
- JavaScript (DOM Manipulation & LocalStorage)
